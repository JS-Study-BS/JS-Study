# 221114_수민

### 1. 다음 코드에서 콘솔에 출력되는 값이 무엇인지 설명하시오.

```javascript
const _ = require('lodash');
const obj = { x: { y: 1 } };
const c = _.cloneDeep(obj);

// 1.
console.log(obj.x === c.x)
// 2.
console.log(obj.x.y === c.x.y)
```

<summary>정답</summary>

<div markdown="1">
  1. false. obj.x와 c.x는 서로 다른 객체를 참조한다.
  2. true. obj.x.y와 c.x.y의 값은 모두 1이다.
</div>
